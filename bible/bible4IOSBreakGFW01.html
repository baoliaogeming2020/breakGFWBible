由于IOS是封闭系统，中国大陆使用iPhone或iPad翻墙有些麻烦。用户只能从苹果商店(App Store)下载翻墙应用程序(App)。随着使用苹果手机翻墙的用户越来越多，防火长城(GFW)也越来越高，能在中国使用的苹果翻墙APP屈指可数。尤其是中国大陆AppStore的限制，更增加了iPad、iPhone用户的翻墙难度，导致下载iOS翻墙软件也很困难。

IOS翻墙圣经

一、必要说明：

IOS翻墙的手机或平板一定不要越狱。根据长期翻墙经验，苹果手机安全排名：美版、非美非大陆版、港版、陆版。因近期香港已经大陆化，港版苹果手机不再是安全翻墙的首选。就算只能买到陆版苹果手机，也比安卓手机安全。
如果经济条件不允许购买新的苹果手机，二手苹果也是不错的选择。但在购买过程中一定要保证能够出厂化(也就是将整个手机重装：设置 -> 通用 -> 传输或还原iPhone -> 抹掉所有内容和设置 -> 重启后等于新手机，要能够用自己的AppleID正常登陆)。这样才能保证购买的手机用美区AppleID登陆，才能下载安全的翻墙软件。
如果有可能，购买二手苹果最好的办法是现场要求出厂化手机，用自己的AppleID登陆。
有战友问，二手苹果以前安装过微信，是否能够购买。
答案是：没问题，可以购买，以前安装过什么软件不重要，因为，你买回来是要出厂化，把所有历史APP都要完全删除。

苹果手机翻墙必要步骤：
  1. 注册美区AppleID：因为，大陆的AppleID很难下载翻墙App，同时，所有数据会进入"云上贵州"，就算你的翻墙软件加密再强也无法保证安全。
  2. 安装翻墙软件："VPN不安全！VPN不安全！VPN不安全！"重要的事说三遍，真正安全的翻墙App包括：shadowrocket(小火箭)、quantumult(圈)。本教程只讲shadowrocket。
  3. 再次注册AppleID：刚才已经注册过了，为什么还要注册。因为，只能保证近期安全(苹果与大陆的合作策略经常变化)，因此，在翻墙成功后要再次注册新的AppleID，抛弃刚才注册的ID，用新ID重新装手机，重新翻墙。
  4. 如何利用VPN：上边说VPN不安全，但小火箭是收费APP，如果注册新ID后，再下载小火箭要再购买一次，所以，如果在第二次注册ID前用VPN翻墙，也可以剩点钱。有大量免费的VPN，有大量的限时免费VPN，所以VPN就不要花钱了。免费的VPN不要卸载，当节点不稳定时，备用。
我的建议步骤：注册美区AppleID -> 登陆 -> 下载VPN -> 翻墙注册新的美区AppleID -> 下载小火箭

    2. 简单翻墙：

    1. 复制节点：复制V2ray节点到剪贴板，全选文本框内的文本并复制；
    2. 返回本软件：会弹出"将复制配置添加到您的列表"窗口，点击"添加"，节点添加成功。可以一次复制多个，一行一个。
    3. 首页->连通性测试：点击此项后，所有节点后会出现时间，如："191ms"，标注此节点连通速度，尽量选择时间短的节点。本测试根据"设置->延迟测试方法"中的协议测试，其中"TCP"用"tcping"命令测试，"ICMP"用"ping"命令测试，"CONNECT"通过访问指定网站测试。软件默认用"TCP"测试，不一定准确。所以，如果时间最短的节点无法连接，多换几个节点试试。有些节点中设置了反向代理，对某些网站访问速度较快，同时，有些节点因IP问题，无法访问某些网站。
    4. 首页->全局路由->代理：设置翻墙全局模式，所有连接都走代理。当设置"代理"模式时，因GFW封锁国外访问国内网站，你可能无法用国内的APP或者无法访问国内网站，因此，每次用完记得关闭。也可以选择"配置"模式。

    五、常见问题：

    1. 如何判断订阅链接的安全性：用浏览器打开链接，看到的文本以"dm"开头就是"vmess"链接，安全性不错，如果以"c3"开头就是"ss"或"ssr"链接，安全性不好。
    2. 自建节点无法连接最可能问题：最可能的是服务器与本机时间不协同，因v2ray要求时间差在2min中内，所以，如果你本机时间是网络同步，就要改服务器节点的时间。也有部分战友本机时间是手动设置，甚至年月都与当前时间不匹配。
    3. 什么是翻墙的最后一道保障：答案是telegram或者i2p。因为telegram有一种mtproto代理，这种免费的独特代理数量及其庞大，好多telegram群为了推广都会不断推出此类代理，同时有大量发布vmess节点的群。i2p，类似tor，但不用获取节点。也有些人说最后一道保障是其它，我解释一下，比如：tor，国内只有联通速度还可以，但难以获取节点。
    4. 自建节点：本教程中的节点不能长期使用，因为节点公布后，常被中共常封锁，最好的办法是自己建节点，油管上有大量的教程，搜索学习才能保证长期翻墙和安全。

    六、节点格式：

    v2ray翻墙节点有6种格式，这些格式一方面方便了节点的传播和添加，但也让大部分战友迷惑。获取错误格式通常会在添加节点时提示错误。
    1. 通用链接格式："vmess://..."，此种格式全部内容都是base64加密，无明文，也就是看不懂的乱码。这种格式适用大部分v2ray软件。
    2. kitsunebi格式："vmess://...?network=tcp&aid=0&tls=0&allowInsecure=1&mux=0&..."，此格式前半部分是base64加密，后边是明文参数，也就是"?&a=b"。此类格式只适用kitsunebi。
    3. BifrostV格式："bfv://45.63.34.187:50750/shadowsocks/1?rtype=lanchina&dns=8.8.8.8&method=..."，此格式全是明文参数，此类格式只适用BifrostV。
    4. v2ray订阅链接：订阅链接内容是对多个vmess再次base64加密的文本以http或https发布，一般都是"https://..."。此种链接里的内容也可能会出现以上三种vmess的格式，大部分都是第一种，也有把SSR加入其中的。
    5. clash订阅链接：clash专用的订阅链接内容不详细说了。这种格式太大了。此格式只适用clash。
    6. 二维码：1 - 3都可以生成二维码，但不同格式生成的码只适用于原格式软件扫码。

    七、墙内传播：

    当你在墙内传播本教程时一定要给接收者一个有效的节点。帮助他完成安全翻墙。给节点时最好的办法是修改此文档中节点部分，然后再加密压缩，这样可降低传播风险(因为，降低了教程被墙识别的可能性)。最好不要单独传播节点文本。
    修改教程内节点方法：
    1. 用文本编辑器打开本教程，会看到一堆代码，不要慌，不是让你学代码，只让你按步骤改些东西就行了。
    2. 找到文档的第 166 行，会看到"<input value="，把value后边双引号内的内容改成你的节点文本就可以了。
    3. 保存 -> 加密压缩 -> 传播。这种方法很安全，中共的识别系统无法对你的这种行为进行监控。
