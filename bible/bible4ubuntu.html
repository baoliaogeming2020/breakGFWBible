以前一直说：用 linux 的战友都是高手，所以我不会写 Linux 翻墙教程。但最近帮一个朋友翻墙，他拿出来个十年前的笔记本电脑，启动 Win10 要十几分钟，决定帮他装个 Linux。最终实现翻墙后发现，难度不是一般的高，但又没有那么高。而且，过往发现有好多小白战友在用 Linux 翻墙。
通过这件事才发现，Linux 翻墙真的不是一般的好用，而且也存在一定的难度，同时也是中共无法封锁的翻墙方式，所以，我决定写一份 Linux 翻墙教程。
如果你是墙内战友，还只有一部手机翻墙，这种翻墙方式也是一种不错的选择。
这种翻墙的成本极低，甚至比第二部手机翻墙的成本还要低。十年前的笔记本电脑够用，但十年前的手机绝对没法用来翻墙。而且用电脑翻墙的灵活性与安全性都比手机强很多。

Linux 翻墙圣经

一、必要说明：

以前一直不敢写这种翻墙的教程，因为，怕写完了，就可能被中共封锁，还不如不写。但Linux 的翻墙，中共还真的不太好封锁，因为一但封锁，可能会对墙内大量 Linux 用户造成巨大的影响。
翻墙有一个矛盾的问题：先翻墙才能够看到翻墙教程。所以，新人学习翻墙比较难。但是，墙内的网络不是一点翻墙教程没有，只不过是很难形成系统的学习。因此，才要写一个成系统的 linux 翻墙教程。帮助小白解决翻墙的第一步。
不敢写太多关于翻墙的内容，因为怕中共把墙内仅有的一点缝隙也封锁。
QV2ray 停止更新，但是，大部分历史教程基本都是关于 Linux 如何用 QV2ray 的翻墙教程，而新近兴起的 V2rayA 已经是 Linux 较好的翻墙工具，但是，能够传入墙内的教程较少。
Linux 的选择是一个复杂的问题，因为，版本太多。大部分 Linux 都是面向服务器的，面向桌面的较少。网上能够全面解释各版本区别的网络文章较少，名字实在是太多，我列出的都不够十分之一。
如：ubuntu debian centos fedora RedFlag SUSE Redhat Gentoo FreeBSD kubuntu xubuntu
同时，同一个 Linux 操作系统又分为不同的版本，如：centOS 根据 CPU 不同分为x86_64 ARM64 (aarch64) IBM Power BE (ppc64) IBM Power (ppc64le) ARM32 (armhfp) i386
好不容易整明白 x86_64 是啥，又分为不同的镜像。好不容易整明白镜像是啥，找了一个国家镜像，进去看到的是 DVD Everything Minimal Netinstall，同时也分成 ISO torrent。还能看到 sha256sum 这种东西。
其实能找到这些复杂的东西，你已经可以下载操作系统了，但是，你是不是基本已经懵逼了。
有多少战友能搞明白应该下载 DVD-ISO。
安装过程要选择 GNome 和 KDE，这是啥？好不容易都安装好了，启动后看到的是全黑界面，跟自己认为的操作系统完全不一样，又有多少战友知道 StartX 是启动桌面的命令。是不是要疯了？？
所以本教程将为战友做出最精简的选择。一切以傻瓜操作为基础。

二、乌班图：

乌班图就是 ubuntu 的中文名，也有好多人叫"友邦兔"的，但不专业。

1. 为什么是乌班图：

  • 以桌面为主，安装过程中基本是"下一步"就可以。
  • 流行程序高，国内用户较多，不太容易完全封锁。
  • 版本相对少，没有选择难题，小白上手非常简单。

    • 2. 安装：

    这部分不会说的太详细，墙内有大量教程。但安装过程中的一点问题会点到。安装后某些命令也会提到。

    下载：

    访问中文官网 > 下载 > Ubuntu桌面系统 > 下载。一般会看到两个版本，其中有一个名称中有"LTS"，就是长期支持版本，也就是稳定版本。一般下载这个版。会得到"ubuntu-22.04.2-desktop-amd64.iso"。这里需要关心的就是 amd64 这个是指 intel 或 amd 的64位 CPU。下边写着安装此系统的"推荐的系统配置要求"，如果你的旧电脑不够这个要求，就去下载以前的版本，因为，版本越低要求电脑配置越低。

    安装：

    这里只推荐 USB 安装。所有教程来自官网教程。这个教程中第2步是下载，前边已经讲过了，第3-5步是安装。看不懂英文的用谷歌翻译。
    第3步：说的是如何制作 USB 启动盘。很简单，根据手头有的安装好的操作系统下载制作 USB 启动盘的软件 balenaEtcher。这个软件非常简单，双击后安装，打开后界面三个按钮，第一个按钮，选择 iOS 包，就是刚才下载的 ubuntu；第二个按钮，选择 U 盘，不用事先格式化，选择时一定要选择 U 盘，记住 U 盘名后边有用；第三个按钮，开始做 U 盘。等待结束就行了。
    第4步：说的是 U 盘启动，大部分笔记本电脑是按开机后直接按 F12 或 Fn+F12。启动后一般会看到 U 盘名(刚才让你记的)，键盘上的上下箭头键选择这个 U 盘，按回车，进入安装界面。
    第5步：开始安装，第一步选择语言，选择中文，后这的安装界面都是中文了。每个界面都读一下，不懂的就按默认选择，或者百度一下安装教程，按界面教程选择是否。就这么简单。

    三、V2rayA：

    这个软件有全命令远程安装过程，所以，不用访问网页直接安装。解决了某些翻墙软件需要先翻墙才能下载的问题。

    安装：

    官网教程如果无法访问，说明需要先翻墙才能看教程，没关系下边会讲，但只讲远程安装。以下每个注释间命令复制并打开终端粘贴，回车，就完成安装。
    1. 安装 V2Ray 内核
    # v2rayA 提供的镜像脚本 curl -Ls https://mirrors.v2raya.org/go.sh | sudo bash # 关掉服务，因为 v2rayA 不依赖于该 systemd 服务 sudo systemctl disable v2ray --now
    2. 安装 v2rayA
    # 添加公钥
    wget -qO - https://apt.v2raya.org/key/public-key.asc | sudo tee /etc/apt/trusted.gpg.d/v2raya.asc # 添加 V2RayA 软件源
    echo "deb https://apt.v2raya.org/ v2raya main" | sudo tee /etc/apt/sources.list.d/v2raya.list
    sudo apt update     # 安装 V2RayA
    sudo apt install v2raya # 启动 v2rayA
    sudo systemctl start v2raya.service # 设置开机自动启动
    sudo systemctl enable v2raya.service
    如果安装过程中提示错误，读一下提示，一般是操作系统缺少软件，解决办法：sudo apt-get install [缺少软件的名称]。
    如：第一步，安装 V2Ray 内核时提示错误，一般是缺少 curl 命令。
    解决办法：sudo apt-get install curl

    使用：

    这里只介绍官网快速上手教程，其它进阶教程去官网查看。
    首先，打开 firefox 浏览器，在地址栏里输入 http://localhost:2017，然后回车(也可以到程序里找 V2rayA 图标点击)，进入创建账号界面。你需要创建一个管理员账号，请妥善保管你的用户名密码。如果自己不想记密码，浏览器会提示是否保存，直接保存。
    然后，进入主界面一个弹出窗口，创建或导入节点，这一步可以关闭窗口，因为，在主界面也可以完成，后边会介绍，或者创建或导入节点。
    最后，进入主界面。如下：

    1. 创建或导入节点
    后边介绍如何在百度中找节点，初期在墙内找到的节点基本都是 "vmess://ew0KICAidiI6ICIyIiwNCiA..." 格式，所以，创建基本用不上，直接粘贴后导入就可以了，会在 "SERVER" 中出现此节点。也可能会找到 http:// 或 https:// 格式的订阅链接，也是粘贴到导入窗口，会在窗口中间 "SERVER" 后出现订阅导入的所有节点标签，以链接命名。
    2. 测试节点
    窗口中间一但有了节点，就需要测试节点是否连通。选择节点左侧的选择框，打对号的为已选择。然后按 ping，会在节点列表的"延时"一列出现此节点的连通时间，时间越短说明节点越快，"TIMEOUT"表示节点不通，但也可以上网试一下，因为 ping 不通，不一定不能上网。
    3. 设置
    本步骤可以忽略，因为，默认设置也可以上网。按图片设置即可。
    4. 选择节点
    选择用于翻墙的节点，可以选择多个。多节点代理设置在窗口左上角的"proxy"按钮，不多介绍，默认设置可用。以后去官网学。
    5. 启动代理
    按窗口左上解的启动按钮，完成代理翻墙。后边还要设置浏览器。

    1. 进入设置
    点 firefox 浏览器窗口右上角三横按钮，进入设置界面，下拉到最底部的网络设置，按设置按钮。
    2. 手动配置代理
    按图片中设置。确定后完成翻墙，用谷歌试一下是否可以访问。如果不能访问，去找新的节点。
    有时重新开机后无法翻墙，解决办法，等待一段时间，或者，清空此项设置后确认，再重新设置。

    四、节点：

    所有的 V2ray 都需要节点才能翻墙，节点就是翻墙用的设置好代理的虚拟机，V2RayA 支持节点格式有：Socks、HTTP、Shadowsocks、VMess、Trojan和VLESS，也就是 ss:// ssr:// http:// vmess:// trojan:// vless://。还有一种将种类节点做成订阅链接的 http:// 或 https:// 格式的节点订阅。这里就不提供节点了，因为提供了也用不了太久(如果你足够细心，可以发现我已经提供了订阅链接)，不过可以教大家方法。
    你可以百度搜索不同的格式，能够搜索到好多节点，甚至订阅链接。不多说了，怕说多了中共会封锁，就搜索不到了。
    注意：这些节点有好多已经过期，有些能翻墙出去，有点耐心。找到能够翻墙的不要用太久，尽快学习自建节点才能保证安全。

    五、其它：

    1、虚拟机：

    如果不想买二手电脑，可以在现有电脑上安装虚拟机跑 ubuntu 翻墙。

    2、进阶：

    如果没有自己建节点的技术能力，可以选择 tor 或者 I2P 试试。墙内 tor 和 I2P 很难实在初次连接，翻墙后可实现，断开翻墙后依然能够连接。
